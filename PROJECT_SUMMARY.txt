================================================================================
DESCRIZIONE PROGETTO
================================================================================
Applicazione web per la valutazione della maturità digitale di aziende e 
pubbliche amministrazioni. Include questionari di livello 1 e 2, generazione 
report PDF, pannello admin, e assistente AI.

================================================================================
ARCHITETTURA
================================================================================
- Frontend: React + TypeScript + TailwindCSS + Vite
- Backend: FastAPI (Python) + SQLAlchemy async
- Database: PostgreSQL (Neon)
- Hosting Frontend: Vercel
- Hosting Backend: Render

================================================================================
PROBLEMI NOTI / DA RISOLVERE
================================================================================

1. Assessment Livello 2 - Errore "Devi completare almeno un assessment di 
   livello 1": La query di verifica potrebbe non riconoscere correttamente 
   gli assessment completati. Verificare che lo status sia "completed" e 
   che il campo level sia 1 o NULL.

2. Assistente AI non funzionante: Richiede API key OpenAI con crediti attivi.
   L'utente ha rinunciato a configurarlo per ora.

3. Backend Render (free tier): Si "addormenta" dopo inattività. Il primo 
   click può richiedere 30-60 secondi per il risveglio.

================================================================================
COMANDI UTILI
================================================================================

# Avviare backend localmente
cd backend
pip install -r requirements.txt
uvicorn main:app --reload

# Avviare frontend localmente
cd frontend
npm install
npm run dev

# Deploy manuale
git add .
git commit -m "descrizione"
git push
# Vercel deploya automaticamente
# Render deploya automaticamente (o Manual Deploy dalla dashboard)
